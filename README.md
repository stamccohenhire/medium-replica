# medium-replica

## Overview
This is a practice project aimed at replicating the Medium blogging platform.

## Features
- User authentication
- Article creation and editing
- Commenting system

## Tech Stack
- HTML, CSS, JavaScript for frontend
- Node.js for backend
- MongoDB for database

## Usage
Clone the repository and follow the installation instructions in the FAQ section. Make sure to set up your environment variables.

## Future Plan
Implement additional features like user profiles and enhanced search functionality.
